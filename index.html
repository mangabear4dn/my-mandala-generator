Welcome to my personalized mandala calculator!

How does it work?

  1. Please select a date you wish to calculate a mandala for (top right corner)!
  2. Click submit!
  3. Get a mandala pie slice!

The mandala pie slice is repeating segment of the mandala. Using this you can create and color your own mandala. Or just analyze the colors by checking out the data in the table.

This pages purpose is to help with the calculations for creating a personalized mandala. But anyone is welcome to use this calculator if they wish.
